SESDC Microgrid Design Tool

For my senior capstone project at Seattle University, I am working on a team of 4 to develop a web application that models small, self-contained renewable energy systems. The joint sponsors of the project are KiloWatts for Humanity and the SESDC branch of IEEE. This fall, I updated the code files to calculate and graph the solar power generated by the system. I also built the basic functionality of the web application using React and JavaScript for the frontend and a Python Flask backend. This base application allows a user to upload a CSV file containing 8760 irradiance values, and returns a graph of the solar power generated.

Statistics for Simulation Based Inference

Under Dr. Fischer, I am conducting research on neurons by simulating electrical activity using an Adaptive Exponential Integrate and Fire neuron model. As a research assistant this fall, I wrote code to extract information about the neuron's firing patterns, for the purposes of using simulation-based inference and a neural netowrk to replicate results. Functions for the model were written in Python using numpy, matplotlib, and scikit-learn. The slides were built using LaTeX.
